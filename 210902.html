<'meta' 태그>

가장 먼저 살펴볼 것은 'meta' 태그입니다.
'meta' 태그는 화면에 출력되지 않는 태그로, HTML 문서에 대한 메타데이터를 정의합니다.
메타데이터는 어느 데이터에 대한 데이터 정보로, 쉽게 생각하자면 그냥 데이터를 설명해주는 데이터라고 보시면 됩니다.
아직은 너무 깊게 파헤치지 않아도 됩니다.
'meta' 태그는 항상 'head' 태그 내부에 쓰여서 문자 설정, 페이지 설명, 키워드 설정, 작성자 등을 정의합니다.
이것들 외에도 많은 역할을 하지만 저는 몇 가지만 살펴볼 것입니다.

먼저, HTML에 사용되는 문자를 정의하는 것인데, 이는 우리가 이미 알고 있으며 심지어 사용하고 있습니다.
바로 'charset' 속성을 통해 정의하는 것입니다.

이렇게 말이죠.
'charset' 속성을 적용하면 웹 브라우저가 해당 HTML이 어떤 문자 인코딩 방식으로 저장되어 있는지 알 수 있게 됩니다.
위와 같이 'charset="utf-8"'로 적용하면 웹 브라우저는 HTML이 'utf-8' 인코딩 방식으로 저장되어 있음을 알게 됩니다.

<의미론적 태그>

의미론적 태그란 구조(structure)를 정의하는 태그입니다.
구조만 정의하는 태그이기 때문에 화면상에는 출력되지 않습니다.
이러한 태그의 종류는 다음과 같습니다.

태그 기능
article 본문 정의
aside 광고와 같이 페이지의 내용과는 관계가 적은 내용들을 정의
details 기본적으로 화면에 표시되지 않는 정보들을 정의
figure 삽화나 다이어그램과 같은 부가적인 요소를 정의
footer 화면의 하단에 위치하는 사이트나 문서의 전체적인 정보를 정의
header 화면의 상단에 위치하는 사이트나 문서의 전체적인 정보를 정의
main 문서에서 가장 중심이 되는 컨텐츠를 정의
mark 참조나 하이라이트 표시를 필요로 하는 문자를 정의
nav 문서의 네비게이션 항목을 정의
section 문서의 구획들을 정의
time 시간 정의
